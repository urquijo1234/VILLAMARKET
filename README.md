# Villamarket Store - Sales Registration

This Java project provides a solution for the Villamarket Store, allowing the registration of up to 10 different products. It utilizes a JTable component to track products added to the sale, automatically calculating the subtotal, VAT, and total amount to pay.

## Requirements

- Java 8 or higher.
- Java-compatible IDE (Eclipse, IntelliJ, NetBeans, etc.).
